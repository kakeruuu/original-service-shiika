# README

概要
俳句（川柳）・短歌を使って気持ちや出来事を投稿できるサービス。

機能一覧
・＃タグ機能
・同じ＃タグを全て表示する機能
・お気に入り登録・解除機能
・お気に入り表示機能
・投稿機能
・投稿削除機能
・ユーザー編集機能
・ユーザー画像投稿機能（carrierwave mini_magick）
・エラーメッセージ日本語化機能(rails-i18n)
・新規登録機能
・認証機能(bcrypt)
・ログイン機能(session)
・ページネーション機能(kaminari)
・MySQL
・フレームワーク(bootstrap rails)
・開発環境(Cloud9 heroku linuxコマンド)
